## Problem
애플리케이션 레이어에서 도메인 ID를 문자열로 변환할 때 Value Object에 from_str 메서드가 없어 임시 workaround 코드를 작성했다.

## Cause
DDD에서 Value Object를 설계할 때 도메인 레이어에서 먼저 모든 ID를 완전하게 정의하지 않고 애플리케이션 레이어 구현 중에 필요성이 발견됐다.

## Rule
DDD Python 구현 시 모든 ID Value Object에는 generate(), from_str() 두 팩토리 메서드를 처음부터 포함시켜라.
